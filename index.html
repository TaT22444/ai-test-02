📋 ステップバイステップガイド

1

Cursorの準備

まず最初に、Cursorエディタをセットアップしましょう。

  • Cursor公式サイトからダウンロード・インストール
  • APIキーの設定（OpenAI、Claude、その他のAIプロバイダー）
  • 基本的な設定の確認

💡 CursorでAIを使う基本的な方法

1. Cmd+K (Mac) または Ctrl+K (Windows) でAIチャットを開く
2. 「Reactでタスク管理アプリを作って」などの指示を入力
3. 生成されたコードを確認・修正
4. Tab キーで提案を受け入れ
2

プロジェクトの作成

AIを使って実際にWebサイトやアプリケーションを作成します。

🤖 AIへの効果的な指示例

「モダンなデザインのポートフォリオサイトを作成してください。
以下の要件を満たしてください：
- レスポンシブデザイン
- ダークモード対応
- スムーズなアニメーション
- SEO対応のHTML構造
- Contact フォーム付き」

✨ コツ

  • 具体的な要件を明確に伝える
  • 技術スタック（React、Vue、Vanillaなど）を指定
  • デザインの参考サイトがあれば共有
  • 段階的に機能を追加していく
3

GitHubリポジトリの作成

作成したプロジェクトをGitHubで管理します。

  1. GitHubにログイン
  2. 「New repository」をクリック
  3. リポジトリ名を入力（例：my-awesome-website）
  4. 「Public」を選択（Netlifyの無料プランを使用する場合）
  5. 「Create repository」をクリック

🔧 ローカルでのGit設定

# プロジェクトディレクトリで実行
git init
git add .
git commit -m "Initial commit: Add project files"
git branch -M main
git remote add origin https://github.com/yourusername/your-repo-name.git
git push -u origin main
4

Netlifyでのデプロイ設定

GitHubリポジトリをNetlifyに連携して自動デプロイを設定します。

  1. Netlifyにログイン（GitHubアカウントで連携推奨）
  2. 「New site from Git」をクリック
  3. 「GitHub」を選択
  4. デプロイしたいリポジトリを選択
  5. ビルド設定を確認・設定
  6. 「Deploy site」をクリック

⚙️ 主要なビルド設定

プロジェクトタイプ Build command Publish directory
静的HTML （空欄） ./
React npm run build build
Vue.js npm run build dist
Next.js npm run build out
5

カスタムドメインの設定（オプション）

独自ドメインを使用したい場合の設定方法です。

  1. Netlifyのサイト管理画面で「Domain settings」を開く
  2. 「Add custom domain」をクリック
  3. 取得済みのドメインを入力
  4. DNS設定でCNAMEレコードを追加
  5. SSL証明書の自動設定を確認

🌐 DNS設定例

Type: CNAME
Name: www (またはサブドメイン)
Value: your-site-name.netlify.app

🔧 トラブルシューティング

ビルドが失敗する

+
  • package.json の dependencies を確認
  • Node.js のバージョンを確認
  • ビルドコマンドが正しいか確認
  • 環境変数が適切に設定されているか確認

サイトが表示されない

+
  • Publish directory の設定を確認
  • ファイルパスの大文字小文字を確認
  • index.html が正しい場所にあるか確認
  • ブラウザのキャッシュをクリア

フォームが動作しない

+
  • Netlify Forms の設定を確認
  • フォームタグに name 属性と netlify 属性を追加
  • hidden input で form-name を指定
  • reCAPTCHA の設定確認
